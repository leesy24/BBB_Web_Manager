SNMP v1/v2/v3 Agent   Help

Enable or disable SNMP(Simple Network Management Protocol) support.
MIB-II (RFC 1213): System, Interface, IP, ICMP, TCP, UDP
MIB-I (RFC 1317): Serial Interface
V1/2 Attribution   Help

Use SNMP V1/2 Attribution ReadOnly or Read/Write or Disable.
SNMP V1/2 Readonly and Read/Write Community name is public
V3    Attribution   Help

Use SNMP V3 Attribution ReadOnly or Read/Write.
V3 Username / Password  /    Help

Set the V3 username & password of the Eddy SNMP Agent.
The Password is at least 8 character string!
please refer to the Eddy User Manual.
TRAP IP / Port  /    Help

Set the IP address and Socket Number of the SNMP TRAP Server.
please refer to the Eddy User Manual.
If IP is set to 0.0.0.0, TRAP feature is disabled.
